A note about system design steps and system implementation steps.

### System Design Steps and System Implementation Steps

系统设计通常包含三个步骤：
```
1. 制定系统接口。
2. 制定系统结构，并列出元素清单。
3. 制定元素接口。
```

系统实现通常包含两个步骤：
```
1. 制作和测试元素。
2. 组装和测试系统。
```
